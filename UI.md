# UI.md

# 画面仕様

## 1. 基本方針

- 一覧画面は現行のホワイトボードの見た目を可能な限り維持する
- 閲覧性を最優先とし、電話応対時に一目で状況を把握できるようにする
- 編集操作は別画面または簡易ダイアログで行う
- 氏名札の白／赤表示を維持する
- 休暇、出張、外出などは磁石札相当の状態表示で表現する

## 2. メイン画面

### 2.1 目的

- 社員全員の現在状況を一覧表示する
- 外線電話応対時の確認を容易にする

### 2.2 画面構成

上部:
- タイトル: 行動予定表
- 日付表示
- 更新ボタン
- 編集ボタン
- 検索ボックス

中央:
- 一覧表

列構成:
- 氏名
- 行先
- 内線番号
- 備考

表示対象:
- 使用中フラグONの社員のみ表示する

下部:
- 最終更新日時
- ロック状態表示
- 利用者名表示

### 2.3 表示ルール

- 氏名欄:
  - 出社中は白背景
  - 退社済みは赤背景
- 所在状態:
  - 氏名欄または行先欄付近に札風ラベルで表示
- 内線番号:
  - 社員マスタから取得した固定値を表示
- 備考:
  - 長い場合は省略表示し、選択時に全文表示可能とする

### 2.4 操作

- 行選択
- 検索
- 編集画面起動
- 画面更新

## 3. 編集画面

### 3.1 目的

- 対象社員の状態を更新する

### 3.2 入力項目

- 氏名
- 内線番号（表示のみ）
- 出社状態
- 所在状態
- 行先
- 備考

### 3.3 操作ボタン

- 保存
- キャンセル
- ロック解除（管理者のみ）

### 3.4 入力制御

- 氏名は選択式
- 氏名選択候補は使用中フラグONの社員のみとする
- 内線番号は編集不可
- 出社状態は必須
- 所在状態は必須
- 行先、備考は任意

## 4. 管理画面

### 4.1 目的

- 社員マスタを管理する

### 4.2 項目

- 氏名
- 所属
- 内線番号
- 表示順
- 使用中フラグ

### 4.3 操作

- 新規追加
- 編集
- 無効化
- 並び順変更
- 使用中フラグOFFの社員も表示し、再有効化できるようにする

## 5. 状態表示案

- 在席
- 外出
- 出張
- 休暇
- 会議
- 直行
- 直帰
- その他

状態は札風に表示し、色または枠線で区別する。

## 6. 画面遷移

- メイン画面
  - 編集画面
  - 管理画面
- 編集画面
  - 保存後メイン画面へ戻る
- 管理画面
  - 保存後メイン画面へ戻る
