# USECASE.md

# 行動予定表システム ユースケース

## 1. 想定利用者

- 電話応対者
- 一般社員
- 管理者

## 2. 主な利用シーン

### 2.1 外線電話応対時に所在を確認する

1. 電話応対者がメイン画面を見る
2. 対象社員の行を確認する
3. 氏名札の色で出社中か退社済みかを判断する
4. 状態札で休暇、出張、外出等を確認する
5. 行先、備考を見て補足を伝える

### 2.2 社員の状態を更新する

1. 更新者が編集開始を行う
2. ロック取得に成功した場合のみ編集可能となる
3. 対象社員を選択する
4. 出社状態、所在状態、行先、備考を更新する
5. 保存する
6. 履歴を保存する
7. ロックを解除する

### 2.3 管理者が社員情報を変更する

1. 管理者が管理画面を開く
2. 社員名、内線番号、表示順などを更新する
3. 保存する

### 2.4 起動時に更新版を確認する

1. アプリ起動
2. `latest.json` を確認
3. 新版があれば更新案内または更新処理を実施

## 3. ユースケース一覧

- UC-01: 社員一覧を閲覧する
- UC-02: 対象社員の状態を確認する
- UC-03: 対象社員の状態を更新する
- UC-04: 編集ロックを取得する
- UC-05: 社員マスタを変更する
- UC-06: バックアップを取得する
- UC-07: 更新版を確認する
