# koudouhyo

社内用の行動予定表を電子化するためのデスクトップアプリ案。

## 目的

- 外線電話応対時に所在をすぐ確認できるようにする
- 現行のホワイトボード運用に近い見た目を維持する
- 氏名札の白／赤表示と状態札を電子化する
- SQLite + 共有フォルダ + 低頻度更新で運用する

## 主要方針

- 見た目は現行ボードを可能な限り維持
- 内線番号は固定値として社員マスタで管理
- 1日数回の更新を想定
- 編集ロックあり
- バックアップあり
- 起動時バージョン確認あり
